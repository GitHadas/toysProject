Toys api documentation

Users

Method

Post

URL

/users

Body

{
"name": "Hadas",
"email": "hadas@example2.com",
"password": "hahahaha1212",
"role": "user", /*not required*/
"date_created": "2023-12-04" /*not required*/
}

Description

Add a new user to the data base.

Method

Post

URL

/users/login

Body

{
"email": "hadas@example2.com",
"password": "hahahaha1212"
}

Description

User login, if the email and password are correct the server will respond with a token.

Toys

Method

Get

URL

/toys

Body

none

Description

Get a list of all the toys, will present 10 results per page.

Method

Get

URL

/toys/search?s=doll

Body

none

Description

search for a toy by searching follwing name or info after the "s=", it will present 10 results per page.

Method

Get

URL

/toys/category/dolls

Body

none

Description

Get a list of all the toys in toys category, it will present 10 results per page.

Method

Post

URL

/toys

Body

{
"name": "cat game",
"info": "cat",
"category": "Fun Category",
"img_url": "https://example.com/toy-image.jpg", /*not required*/
"price": 29.99
}

Description

Add a new toy to the data base, You need a token for this query.
